# 05-01-PLAN: Styling Refinements & Interactions

## Description
This plan updates the entire application's UI to be more visually appealing, consistent, and responsive. It introduces a refined "equestrian-modern" theme.

## Context
Phase 5: UX/UI Polishing. The app is functional but looks basic.

## Tasks
- [ ] **Task 1: Global Theme & Typography** — Update `:root` variables in `index.css` with a cohesive color palette and set up base fonts.
- [ ] **Task 2: Setup Screen Refinement** — Update `SetupScreen.css` for a more inviting layout and better input styling.
- [ ] **Task 3: Dashboard & Player Cards** — Enhance `Dashboard.css` and `App.css` to make player cards more dynamic and responsive.
- [ ] **Task 4: Modals & Overlays** — Style `TransferModal.css` for better focus and clarity.
- [ ] **Task 5: Transaction Log & History** — Polish the log feed and history list for readability.
- [ ] **Task 6: Animations & Transitions** — Add CSS transitions for button hovers and layout shifts.

## Verification
- [ ] App is fully responsive in DevTools (Mobile S, M, L).
- [ ] Colors and fonts are consistent across all screens.
- [ ] Transitions work correctly when opening/closing modals and switching screens.
